Detailed Description

A queue object. Has pointer to last and first elements in the queue. Also hold a counter on the number of element, this is no an obligatory property, but it allows better dubging and can be useful in other cases. The queue is considered valid when count and first and last element state are matching. That is, if count equals to zero, both first and last element must point to NULL. Overwise, when count is one, both first and last element must point to the same element. Otherwise, when count is gretaer than one, each has to point to another element.

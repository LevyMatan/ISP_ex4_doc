Ex-4: Client

The client will be the interface between the user and the server process.
Without any user interaction, the client will attemp to connect tot the Server. Once connected the server will "guide" the client which will display the information to the user.
Collect user input, and validate it correctness, and communicate it back to the server over TCP/IP.

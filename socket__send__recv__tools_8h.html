static

SOCKET_receive_string() uses a socket to receive a string, and stores it in dynamic memory.

Accepts:

p_output_string - a pointer to a char-pointer that is initialized to NULL, as in: 

char *p_buffer = NULL;
SOCKET_receive_string( &p_buffer, ___ );

a dynamically allocated string will be created, and (*p_output_string) will point to it.

sd - the socket used for communication.

Returns:

TRNS_SUCCEEDED - if receiving and memory allocation succeeded TRNS_DISCONNECTED - if the socket was disconnected TRNS_FAILED - otherwise

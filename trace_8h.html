◆ DEBUG_VERBOSITY_LEVEL

#define DEBUG_VERBOSITY_LEVEL   (5)

Debug Levels Description: To enable debug printing, with easier toggle on/off of specific verbosity levels. Each DEBUG_PRINT statment will be assosicated with a specific verbosity level. A global DEBUG_VERBOSITY_LEVEL will define how many prints of logs we wish to see. The higher it is define, the more prints should be shown. To fully utilize this mechanism, each print statement should think out the how needed are those prints. For example, DEBUG_PRINT_FUNCTION_START() assigns level one, because it is useally usefull to have a function start and end prints - lets you know a specific function was called and executed. As a rule of thumb, if the print statment is located in a "deep" or long loops, it is wise to assign a larger debug level, so it will show up on when needed to avoid clutter.
